## How to Run

1. Clone the repository:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Navigate to the project folder:

```bash
cd seu-repositorio
```

3. Install dependencies with npm:
```
npm install
```

4. Run the project:
```
npm run dev
```
